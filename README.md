# TanStack Excel-like Data Table

A React TypeScript application demonstrating an Excel-like large data table.

### Generate New Data

```bash
# Generate default 500 entries
npm run generate-data

# Generate specific number of entries
npm run generate-data 250
```
## Available Scripts

### Development
```bash
npm start          # Start development server
```

### Data Generation
```bash
npm run generate-data [count]  # Generate strain data
```
